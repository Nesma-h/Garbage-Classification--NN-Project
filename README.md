# NN-Project

A neural network project implemented in Jupyter Notebooks.

## Overview

This repository contains Jupyter Notebook implementations of neural network models and experiments. The project primarily uses Python with interactive notebook-based workflows for research, development, and experimentation.

## Project Contents

- **Jupyter Notebooks**: Interactive Python notebooks for exploring neural network architectures and training models
- **Python Scripts**: Supporting utilities and modules

## Technology Stack

- ![Python](https://img.shields.io/badge/Python-3.x-blue)
- ![Jupyter Notebook](https://img.shields.io/badge/Jupyter-Notebook-orange)

## Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook or JupyterLab
- Required Python packages (see requirements.txt or notebooks for dependencies)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/Nesma-h/NN-Project.git
   cd NN-Project
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Start Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## Usage

Open any `.ipynb` file in Jupyter Notebook to view and run the neural network experiments and implementations.
